Airbnb Data Analysis Project

Overview -
This project provides an analysis of Airbnb listings, focusing on key metrics such as price distribution, bedroom counts, and annual revenue trends. Interactive visualizations are developed using Tableau Public to enable insightful exploration of the data.

Visualizations-
Price per ZIP Code – Visualizes average listing prices across different geographic areas.

Distinct Bedroom Listings – Shows the count of listings categorized by the number of bedrooms.

Price by ZIP Code – Compares average listing prices across ZIP codes.

Average Price per Bedroom – Displays the average cost for each bedroom type.

Yearly Revenue Trends – Illustrates weekly revenue trends throughout the year.


Tool Used-
Tableau
